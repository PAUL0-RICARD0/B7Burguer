# B7 • Burguer

Landing page de uma hamburgueria fictícia com vitrine de produtos, busca em tempo real e layout responsivo. Projeto desenvolvido durante a formação Full Stack pela **B7Web**.

---

## Funcionalidades

- **Busca em tempo real** — filtra os burguers pelo nome enquanto digita
- **Filtro por categoria** — Tradicional, Premium ou Vegano
- **Menu hambúrguer** — navegação responsiva para dispositivos móveis
- **Badge de Promoção** — destaque visual nos produtos em oferta
- **Layout responsivo** — adaptado para desktop, tablet e mobile

## Tecnologias

| Tecnologia | Uso |

## 📱 Responsividade

| Breakpoint | Layout |
|---|---|
| `> 820px` | 3 colunas no grid de produtos |
| `≤ 820px` | 2 colunas + menu hambúrguer ativo |
| `≤ 500px` | 1 coluna |

## O que aprendi

- Manipulação do DOM com `querySelectorAll` e `getAttribute`
- Filtros combinados (busca por texto + seleção de categoria)
- Organização de CSS com variáveis e separação de responsabilidades
- Técnica de `margin-top` negativo para sobreposição de seções
- Responsividade real sem frameworks
